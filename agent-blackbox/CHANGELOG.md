# Changelog

All notable changes to Upmarto publishable packages are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-19

### Added

- **upmarto-sdk** (Rust): v1 API client with queue, retry, bootstrap, and session helpers.
- **upmarto-cli** (Rust): `init`, `track`, `workflow`, `session`, `explain`, `flush`, `demo`.
- **@upmarto/sdk** (TypeScript): Node and browser client for `POST /event` and offline queue.
- **@upmarto/cursor**: Cursor IDE hooks for passive event capture.
- **upmarto-vscode**: VS Code extension for passive event capture.
- Release foundation: LICENSE, per-package README files, and distribution metadata.

[0.1.0]: https://github.com/mertcaliskanlinux/upmarto/releases/tag/v0.1.0
